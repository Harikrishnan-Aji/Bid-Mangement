Advanced Python


This Python training course leads students from the basics
writing and running Python scripts to more advanced
features such as file operations,regular expressions, working
with binary data, and using the extensive functionality of
Python modules. Extra emphasis is placed on features unique
to Python, such as tuples, array slices, and output formatting.
